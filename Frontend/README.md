# TCG Marketplace

Marché en ligne de cartes à collectionner avec système d'enchères en temps réel.

## Technologies
- **Frontend** : React.js, React Router v6
- **Backend** : Node.js, Express.js, SQLite (sqlite3), JWT

## Installation

### Frontend
```bash
npm install
npm start
```

### Backend
```bash
cd Backend
npm install
node server.js
```

## Lancement
- Frontend : http://localhost:3000
- Backend  : http://localhost:3001
